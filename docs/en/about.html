About



JSBSim   is an open-source project created by John Berndt (sourceforge...jberndt) to model aerospace vehicles. The project can be found at SourceForge here and at GitHub here.


Cesium   is a company providing mapping solutions, including CesiumJS, an open-source JavaScript library for adding mapping and world-building for other open-source projects. Here is the company's website. The open source project, CesiumJS, has it's own website.


Qt   is a company providing cross-platform software tools for rapid development of applications. They have an extensive C++ based API and an IDE called QtCreator. Their website. Qt is free to use for GPL licensed projects.


PLIB   is the Portable game LIBrary, built by Steve Baker (sourceforge...sjbaker). The project can be found at SourceForge here. QtJSBSim uses the joystick tools from PLIB.


QFlightInstruments   is an open-source project created by Marek Cel (sourceforge...mcel) to model aircraft flight instruments using Qt and SVG graphics. The project is hosted at SourceForge here. QtJSBSim uses the Primary Flight Display from the QFlightInstruments project.


FlightGear   is an open-source flight simulation framework project created by Curt Olson (sourceforge...curt). The project is hosted at SourceForge here. QtJSBSim can run as an "external FDM" for FlightGear, sending flight dynamics data to FlightGear, which can then be used as a visualizer (and much more!).
